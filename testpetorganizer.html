How to use — Neopets Pet Organizer

      


        

          
Getting started

          
    
            
  • Open .md — load a previously saved file to pick up where you left off.
    • 
            
  • Paste from markdown/lookup — paste a raw markdown block or a Neopets userlookup HTML to import an account's pets.
    • 
            
  • The app opens with example data so you can explore before loading your own.
    • 
          

        


        

          
Accounts

          
    
            
  • Each column on the Accounts tab is one Neopets account (max 19 pets).
    • 
            
  • Drag pets between account columns to move them.
    • 
            
  • Click the color dot in a column header to change the account's color.
    • 
            
  • Click the small italic label (or + add label) beneath the account name to add a short note like "main" or "spooky side".
    • 
          

        


        

          
Categories

          
    
            
  • Switch to the Categories tab to see pets grouped by category.
    • 
            
  • Drag pets between category columns to assign or reassign them.
    • 
            
  • Click ✏ to rename a category; click the color dot in the legend or column header to recolor; click the trash icon to delete.
    • 
            
  • Add a new category with the Add Category button at the bottom of the tab.
    • 
          

        


        

          
Tags

          
    
            
  • Right-click any pet card to open the tag popover for that pet.
    • 
            
  • Check/uncheck tags to assign them; type a name and press + or Enter to create a new tag.
    • 
            
  • Click the color dot next to a tag name to change its color.
    • 
            
  • Click the trash icon next to a tag to delete it globally from all pets.
    • 
          

        


        

          
Matrix

          
    
            
  • The Matrix tab shows every pet laid out by account (rows) × category (columns).
    • 
            
  • Drag pets within the matrix to move them to a different account and/or category at once.
    • 
            
  • Hover a pet to see its tags in a tooltip.
    • 
            
  • Click a category or tag chip in the legend above the matrix to highlight matching pets — click multiple to multi-select, and toggle any / all to control how they combine.
    • 
            
  • Click the color dot next to an account name in the matrix to change its color.
    • 
          

        


        

          
Saving & exporting

          
    
            
  • Save Changes — saves directly back to your open .md file (asks to confirm overwrite or rename if needed).
    • 
            
  • Export — shows the markdown text in the panel below the header; you can read it, copy it, or save it manually.
    • 
            
  • Copy — copies the markdown straight to clipboard.
    • 
            
  • The app auto-saves after most changes when a file is already open.
    • 
          

        


        

          
File format

          
    
            
  • Files are plain .md with three sections: ## Accounts, ## Categories, ## Tags.
    • 
            
  • Each section uses name | #color headers with - petname lines beneath. Accounts also support a third field: name | #color | label.
